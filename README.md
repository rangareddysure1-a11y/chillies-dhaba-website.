# chillies-dhaba-website.
"Marketing website for Chillies Family Dhaba."
Chillies Family Dhaba - Official Website

This repository hosts the official static marketing website for Chillies Family Dhaba. The site is designed to be fully responsive (adapts to all devices) and displays the current menu, location details, and direct links for ordering via Swiggy/Zomato.

The entire website is a single-page application built using HTML, Tailwind CSS, and plain JavaScript.
